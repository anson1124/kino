Message format

FRAME			OPT	CONTENT							DESCRIPTION
(of n total)	
======================================================================
0					Sender socket ID				ID of the sending socket (not ROUTER)
1					[0]
2				*	Router ID						0-n ROUTER socket IDs added by every ROUTER, forwarding the message
..
n-7					[0]
n-6					Distribution pattern			One of the following distribution patterns {Unicast, Broadcast}
n-5					Version							Message version
n-4					TTL								Message Time-to-Live in (msec)
n-3					Message ID						Message ID, which is used to find the Worker
n-2					[0]
n-1					Body							Message content

[Version] & [Message ID] are unique identifiers of the message
