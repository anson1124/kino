Message format

FRAME	CONTENT							DESCRIPTION
======================================================================
0		Sender socket ID				ID of the sending socket (not ROUTER)
1		[0]
2		Router ID						0-n ROUTER socket IDs added by every ROUTER, forwarding the message
..
n
n+1		[0]
n+2		Distribution pattern			One of the following distribution patterns {Unicast, Broadcast}
n+3		Version							Message version
n+4		TTL								Message Time-to-Live in (msec)
n+5		Message ID						Message ID, which is used to find the Worker
n+6		[0]
n+7		Body							Message content

[Version] & [Message ID] are unique identifiers of the target Worker
